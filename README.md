# Cafe Gravity
A program for cafe cost calculations.
یک برنامه ساده برای کافه بردگیم که بر اساس هر ساعت بازی، مبلغ مشخصی دریافت میکنند
نرخ پایه به ازای هر ساعت بازی رو وارد کرده، اگر کافه برای اعضای ویژه حساب ایجاد میکنه؛ سوال میپرسه و سپس مقدار ساعت یا دقیقه‌ای که بازی کردید رو وارد و هزینه رو نهایی رو مشخض میکنه
این برنامه رو بر اساس یکی از کافه های شهرمون درست کردم؛ چون به صورت دستی هزینه بازی رو به ازای هر نفر و هر ساعت و دقیقه بازی مشخض میکرد
